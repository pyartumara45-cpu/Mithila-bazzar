# Mithila Bazzar 🛍️

Janakpur ke local kaarigar dwara banaye gaye Mithila print kapde ka online bazaar.

### Hum kya bechte hain?
- **Mithila motif wale kapde** - Kurta, kurti, saree, t-shirt, dupatta, gamcha
- **Hand-painted aur block print** dono option
- **Artist ko direct payment**, beech me koi dalal nahi
- Janakpur ki kala aur sanskriti ko promote karna

### Products
1. **Men**: Kurta, t-shirt, cotton shirt with Mithila design
2. **Women**: Kurti, saree, dupatta, suit set
3. **Unisex**: Gamcha, stole, fabric by meter
